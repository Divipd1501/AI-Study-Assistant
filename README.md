# AI Study Assistant

An AI-powered study assistant developed using Flask and Google Gemini API.

## Features

* User Registration and Login
* AI Question Answering
* AI Quiz Generation
* Quiz Module
* PDF Notes Download
* Study History Tracking
* SQLite Database
* Responsive UI

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript
* SQLite
* Google Gemini API
